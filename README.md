## 📀 Function
1. 로그인 / 로그아웃 / 회원가입
2. 바코드를 찍어서 상품을 조회하고 판매
3. 판매한 기록들을 당일 또는 기간별로 조회
4. 상품정보를 관리할 수 있음 (입력, 조회, 수정, 삭제)
5. 현재 시간 출력
